# lacli
Linear Algebra in your CLI yayo!

## Design:
LAcli is a software that run in your CLI, providing a set of linear algebra commands.
All linear algebra operations are performed using [NumPy](https://github.com/numpy/numpy)

### I/O
You can set the stdin and stdout to use files, pipes or both in awesome combinations.

### Supported file types:
- json
- csv
